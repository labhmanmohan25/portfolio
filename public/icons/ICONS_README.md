# AI & Gen AI skill icons

Icons in this folder are used for the **AI & Gen AI** skills section. Add SVG files with these **exact filenames** so each skill shows its own icon. If a file is missing, the section falls back to `ai.svg`.

| Skill | Filename to add |
|-------|-----------------|
| Generative AI | `openai.svg` |
| Prompt Engineering | `prompt-engineering.svg` |
| RAG | `rag.svg` |
| Vector Databases (Pinecone, Qdrant) | `vector-db.svg` |
| LLM Integration (OpenAI, Gemini, Ollama) | `llm.svg` |
| Workflow Automation (n8n, Zapier) | `workflow.svg` |
| TTS/STT (Whisper, Piper, ElevenLabs) | `tts.svg` |
| mem0 | `mem0.svg` |
| Ollama | `ollama.svg` |
| OpenClaw (...) | `openclaw.svg` |
| Claude Code | `claude.svg` |

**Suggested sources (search “icon svg” or “logo svg”):**
- **openai.svg** – OpenAI logo (openai.com)
- **llm.svg** – Generic LLM / chatbot icon
- **rag.svg** – Document + AI or retrieval icon
- **vector-db.svg** – Database or vector icon (e.g. Pinecone, Qdrant)
- **ollama.svg** – Ollama logo (ollama.ai)
- **claude.svg** – Anthropic / Claude logo
- **prompt-engineering.svg** – Prompt / text cursor icon
- **workflow.svg** – Zapier or n8n logo, or generic workflow
- **tts.svg** – Microphone / audio / ElevenLabs-style icon
- **mem0.svg** – mem0 logo if available, else generic memory icon
- **openclaw.svg** – OpenClaw logo if available, else generic tool icon

`ai.svg` is already included as the default when a specific icon is missing.
